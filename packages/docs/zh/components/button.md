### API

| 属性名   | 类型                                             | 默认值    | 说明 | 版本 |
|----------|--------------------------------------------------|-----------|------|------|
| variant  | `default`、`primary`、`success`、`warning`、`danger` | `default` | -    |      |
| size     | `small`、`medium`、`large `                        | `medium`  | -    |      |
| disabled | `boolean`                                        | `false`   | -    |      |
| ghost    | `boolean`                                        | `false`   | -    |      |
| loading  | `boolean`                                        | `false`   | -    |      |
